# 🧬 PyLadies Bioinfo - Landing Page

Bem-vinda ao repositório oficial do site da comunidade **PyLadies Bioinfo**!  
Este projeto é o principal ponto de contato, divulgação de projetos e conexão para mulheres na intersecção entre biologia, tecnologia e análise de dados.

## 💻 Tecnologias Utilizadas

- **[Next.js](https://nextjs.org/)**: React Framework
- **[TypeScript](https://www.typescriptlang.org/)**: Tipagem estática
- **[Tailwind CSS](https://tailwindcss.com/)**: Estilização e responsividade
- **[Framer Motion](https://www.framer.com/motion/)**: Animações suaves
- **[Swiper](https://swiperjs.com/)**: Carrossel interativo de depoimentos
- **[React Icons](https://react-icons.github.io/react-icons/)**: Ícones padronizados e leves
- **[FormSubmit](https://formsubmit.co/)**: Integração de formulário de contato sem backend

## 🧩 Estrutura do Site

O layout foi componentizado para facilitar manutenção e escalabilidade. As principais seções incluem:

- **Navbar:** Navegação responsiva com menu sanduíche e rolagem suave.
- **Hero Section:** Apresentação da missão da comunidade.
- **Sobre Nós:** Accordion detalhando pilares e valores do grupo.
- **Projetos:** Cards dinâmicos com efeitos visuais.
- **Equipe:** Grid de perfis com links para LinkedIn e Instagram.
- **Depoimentos:** Carrossel interativo com paginação personalizada.
- **Contato & Footer:** Formulário funcional e links úteis.

## 🚀 Como Rodar o Projeto Localmente

1. Clone este repositório:
    ```bash
    git clone https://github.com/pamis-costa/pyladies-bioinfo.git
    ```
2. Instale as dependências:
    ```bash
    npm install
    ```
3. Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4. Acesse `http://localhost:3000` no navegador.
